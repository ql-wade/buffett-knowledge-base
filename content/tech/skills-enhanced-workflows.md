# Skills 增强工作流
# 基于 WorkBuddy 插件的增强工作流

## 一、数据分析工作流

```
用户请求
    ↓
[kev 调度]
    ↓
[dash + data-analysis] ─→ 数据清洗、分析
    ↓
[sage + deep-research] ─→ 深度洞察
    ↓
[doc + document-skills] ─→ 生成报告
```

### 使用场景
- 财务报表分析
- 市场数据研究
- 用户行为分析

### 触发词
- "数据分析"
- "生成报表"
- "深度分析"

---
