# 第11章 商业模式（Business Model）

## 核心观点

### 选择标准：匹配客户购买行为和支付方激励

不是"我想用什么模式"，而是"客户习惯怎么买"和"支付方愿意怎么付"。最好的商业模式是最小化销售摩擦的模式。

### 四种商业模式对比

| 模式 | 收入特征 | 适合场景 | 代表产品 |
|------|---------|---------|---------|
| **Sell Products** | 一次性销售 | 标准化器械 | 支架、人工关节 |
| **Sell Disposables** | 持续性耗材收入 | 高频使用 | 手术刀片、透析耗材 |
| **Rent Equipment** | 租赁+服务 | 大型设备 | MRI、手术机器人 |
| **Service Model** | 按服务收费 | 需持续支持 | 远程监测+诊断服务 |

### Recurring Revenue 最受投资人青睐

在医疗器械领域，**耗材模式（Razor-and-Blade）**是最受追捧的——设备一次性卖出去锁定客户，然后通过耗材持续赚钱。投资人对 recurring revenue 的估值通常是 one-time sale 的 3-5 倍。

## 操作方法

### 商业模式选择矩阵

| 判断条件 | 推荐模式 |
|---------|---------|
| 产品是耗材，使用频率高 | Sell Disposables |
| 产品是耐用设备，单次使用 | Sell Products |
| 产品成本高，医院不愿购买 | Rent Equipment |
| 产品需要持续技术支持 | Service Model |
| 产品有"锁定"特性（耗材+设备） | Razor-and-Blade |

### 定价策略

1. **成本加成法**：成本 × (1 + 目标毛利率)
   - 医疗器械行业目标毛利率通常 > 70%

2. **价值定价法**：基于产品为客户创造的价值定价
   - 例：如果产品每天为医院节省 $500，年节省 $182,500 → 定价 $100K/年仍有吸引力

3. **竞争定价法**：参考竞品价格
   - 只在差异化明显时才用溢价策略

### 单位经济模型（Unit Economics）

```
每台手术/每位患者：
  收入 = 产品售价
  成本 = COGS + 销售成本 + 分摊的研发/管理
  毛利 = 收入 - COGS（目标 > 70%）
  净利 = 收入 - 所有成本（目标 > 20%）

如果毛利 < 50% → 商业模式可能有问题
```
