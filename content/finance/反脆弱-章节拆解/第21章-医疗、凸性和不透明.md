# 第21章 医疗、凸性和不透明

## 章节定位

### 一句话定位
> 第21章批判过度医疗——医源性损伤告诉我们：医疗干预也有"凸性"，小病不治可能比治更好。

---

## 核心观点

### 医源性损伤

**定义**：医疗本身造成的伤害
**例子**：
- 抗生素滥用
- 过度检查
- 手术风险

### 医疗的凸性

```
┌─────────────────────────────────────────────┐
│              医疗干预的凸性                   │
├─────────────────────────────────────────────┤
│                                             │
│  小病不治 → 自己好 → 无损伤                  │
│  小病大治 → 可能有损伤 → 医源性              │
│  大病治疗 → 风险vs收益 → 需要权衡            │
│                                             │
│  关键：区分"自愈"和"需要治"                  │
└─────────────────────────────────────────────┘
```

### 健康策略

- 相信身体自愈
- 小病不急就医
- 大病找正规医疗
